PRD

개요

공공직군 행정업무 슈퍼앱은 팀 단위 행정업무에서 반복적으로 발생하는 일정 공유, 엑셀 처리, 민원 대응, 행정 뉴스 모니터링을 하나의 업무 포털에서 처리하기 위한 내부용 웹 애플리케이션이다.

목표

  • 팀원 휴가, 근무, 출장 일정을 한 화면에서 공유한다.
  • 반복적인 엑셀 분리, 병합 업무를 웹에서 자동 처리한다.
  • 민원 매뉴얼 기반으로 대응 방향과 응대 스크립트 초안을 생성한다.
  • 공공 행정 관련 뉴스를 매일 아침 자동 수집해 업무 참고 자료로 제공한다.

사용자

사용자주요 목적
팀장팀원 일정 파악, 업무 공백 확인, 주요 이슈 확인
팀원본인 일정 등록, 타 팀원 일정 확인, 엑셀 업무 자동화
민원 담당자민원 매뉴얼 기반 응대 스크립트 작성
행정 담당자관련 뉴스 확인, 정책 및 행정 이슈 모니터링
시스템 운영자데이터 백업, 실행 상태 점검, 오류 대응

핵심 기능

팀원 스케쥴 관리

팀원들의 휴가, 근무, 출장, 교육, 재택 등 일정을 캘린더 형태로 등록하고 공유한다.

  • 월간, 주간, 일간 캘린더 뷰
  • 일정 등록, 수정, 삭제
  • 팀원별 필터와 일정 유형별 필터
  • 오늘 일정과 이번 주 주요 일정 요약

엑셀 업무 자동화

사용자가 업로드한 엑셀 파일을 특정 컬럼 기준으로 나누거나, 여러 엑셀 파일을 하나로 합친다.

  • 엑셀 파일 업로드
  • 헤더 분석 및 컬럼 목록 표시
  • 특정 컬럼 값 기준 파일 분리
  • 동일 헤더 파일 병합
  • 처리 결과 다운로드

민원 대응 챗봇

첨부된 민원 매뉴얼을 기반으로 민원 내용에 대한 대응 방향, 확인 사항, 응대 스크립트 초안을 작성한다.

  • 민원 매뉴얼 업로드 및 관리
  • 민원 내용 입력
  • 관련 매뉴얼 검색
  • 민원 요약, 확인 사항, 대응 방향, 응대 스크립트, 참고 근거 제공

뉴스 기사 수집

공공 행정, 지방행정, 복지, 안전, 인사, 예산 등 업무 관련 뉴스를 매일 아침 자동 수집한다.

  • 키워드 기반 뉴스 수집
  • 제목, 언론사, 발행일, 링크, 요약, 키워드 저장
  • 최신 뉴스 목록과 키워드 필터
  • 중복 기사 저장 방지

Operation

Frontend

PowerShell에서는 npm 대신 npm.cmd 사용을 권장한다.

cd frontend
npm.cmd install
npm.cmd run dev

Backend

Backend는 반드시 uv를 사용한다. uv 기본 캐시 경로 접근 오류를 피하기 위해 프로젝트 로컬 캐시를 지정한다.

cd backend
$env:UV_CACHE_DIR="..\.uv-cache"
uv venv .venv --allow-existing
uv pip install fastapi --python .venv\Scripts\python.exe

DB

초기 DB는 SQLite 파일 기반으로 운영하며 권장 위치는 backend/data/app.db이다.

현재 환경에서는 시스템 sqlite3 CLI가 없어도 Python 내장 sqlite3 모듈로 SQLite 사용이 가능하다.

자주 발생하는 에러

PowerShell에서 npm 실행 오류

npm.cmd install
npm.cmd run dev

uv 캐시 접근 권한 오류

$env:UV_CACHE_DIR="..\.uv-cache"
uv venv .venv --allow-existing

FastAPI는 설치됐지만 서버 실행이 안 됨

uvicorn 미설치, app/main.py 미작성, 실행 위치 오류를 확인한다.

cd backend
$env:UV_CACHE_DIR="..\.uv-cache"
uv pip install uvicorn --python .venv\Scripts\python.exe

운영 체크리스트

  • FE 의존성 설치 여부 확인
  • BE .venv 생성 여부 확인
  • FastAPI 설치 여부 확인
  • uv 캐시 경로 설정 확인
  • SQLite DB 파일 백업 확인
  • 업로드 파일 저장소 용량 확인
  • 뉴스 수집 실패 로그 확인
  • 민원 매뉴얼 최신본 반영 여부 확인
